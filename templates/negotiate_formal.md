# 正式协商模板 — 跟进/升级

## 适用场景
首次友好协商未得到回复（24-72小时后），发送更正式的请求。

## 私信/留言模板

### 正式跟进版

```
你好，我是 [姓名]。

我之前曾联系你，请求删除/修改你发布内容中涉及我个人信息的部分，但尚未收到回复。

根据《中华人民共和国个人信息保护法》第四十七条，当个人信息的处理目的已实现或不再必要时，个人信息处理者应当主动删除个人信息；个人信息处理者未删除的，个人有权请求删除。

希望我们能友好协商解决此事。如在 3 个工作日内未得到回复，我可能需要通过平台官方渠道和相关监管机构进行处理。

感谢你的理解与配合。

[姓名]
[联系邮箱]（可选）
```

### 严重泄露版

```
你好，

你发布的内容包含了我的敏感个人信息（[身份证号/银行卡号/手机号/家庭住址]），这已构成对我个人信息权益的严重侵害。

依据以下法律规定：
- 《个人信息保护法》第四十七条：个人有权请求删除个人信息
- 《民法典》第一千零三十四条：个人信息受法律保护
- 《网络安全法》第四十四条：任何个人和组织不得窃取或以其他非法方式获取个人信息

请在收到本消息后 24 小时内删除上述内容。否则我将：
1. 通过平台举报渠道正式投诉
2. 向国家互联网信息办公室（12377.cn）举报
3. 保留进一步追究法律责任的权利

本消息已截图留证。

[姓名]
```

## 使用说明

1. 正式跟进版用于首次友好协商未回复的情况
2. 严重泄露版仅用于身份证、银行卡等严重隐私泄露场景
3. 发送前确保已截图保存证据
4. 3 个工作日内如仍未回复，升级到平台举报
