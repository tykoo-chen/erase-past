# 友好协商模板 — 首次联系

## 适用场景
首次联系内容发布者，请求删除/修改包含个人信息的内容。

## 私信/留言模板

### 通用版

```
你好，我是 [姓名]。

注意到你发布的内容中包含了我的个人信息（[具体说明：如手机号/住址/真实姓名等]），这些信息的公开给我带来了一些困扰。

能否麻烦你帮忙删除/修改一下？非常感谢你的理解！🙏

如有任何问题可以直接回复我。
```

### 简历/文档类

```
你好！

我发现你上传的文档中包含了我的个人简历/联系方式，这份资料已经过时了，而且我不希望这些信息继续公开。

能否帮忙删除这份文档？或者如果方便的话，至少帮忙把我的联系方式（手机号/邮箱/地址）删掉？

非常感谢！
```

### 帖子/讨论类

```
你好，关于你发布的 [帖子/文章标题]，里面提到了我的一些个人信息。

我理解你可能没有恶意，但这些信息的公开确实给我带来了困扰。能否帮忙编辑一下，把涉及我个人信息的部分删除？

谢谢你的理解！
```

## 使用说明

1. 根据实际情况选择合适的版本
2. 替换 `[姓名]` `[具体说明]` 等变量
3. 语气要友好，不要带威胁性
4. 发送后等待 24-72 小时
5. 如无回复，使用 `negotiate_formal.md` 模板跟进
